Basics

At basic level, Container and its items

basic container class:


.container {
    display: flex; //will layout horizontally by default
}
Main Axis and Cross Axis

will lay out horizontally by default... so Main Axis is from left to right, and Cross Axis is top to bottom.

we can swap this by changing the class to:


 .container { 
    display: flex; 
    flex-direction: column; //will layout vertically
}

***this makes Main Axis top to bottom and Cross Axis left to right***

Justify Content

used for aligning content on MAIN AXIS

use property justify-content:

  • justify-content: flex-start; justifies to start of flex container - DEFAULT
  • justify-content: flex-end; justifies to end of flex container
  • justify-content: center; centers content
  • justify-content: space-around; even space around items, half size at beginning and end of container
  • justify-content: space-between; even space around items, but first and last items have no space at beginning and end of container
  • justify-content: space-around; even space around items, including beginning and end of container

**Â  This sets the justification for the container as a whole -- all child items.

Positioning Single Items

.targetedItem {
    margin-left: auto; //will move item and all following items as far right as possible
    margin-right: auto; //will move item and all preceeding items as far left as possible
}

can be used in conjunction with justify-content in order to create more complex layouts

The Flex Property

allows for responsive resizing of items in the container


.targetedItem(S){
    flex: 1; //will grow all items evenly across container
                    
    //actually a shorthand for the following 3 properties: (setting a flex of 1 is also flex: 1 1 0)
    flex-grow: 1; //how much extra space should be allocated to the items.  multiple items with 1 will divide space evenly.  
                  //Item at 1, item at 2, the 2 will get twice as much. items with flex-grow: 0, get nothing.
    flex-shrink: 1; //rate at which items shrink.  in proportion... like grow.  
    flex-basis: 200px; //sets base width of the element.  items will stay at that width if enough space, will flex smaller if needed.
    flex-basis: 50%; //can also be in percent of width of flex container.
                    
}
Aligning Items on Cross Axis

by default, items stretch across cross axis

  • align-items: stretch; DEFAULT VALUE, items stretch across cross axis
  • align-items: flex-start; items pushed to start of container, take up as much space as needed to display content
  • align-items: flex-end; items pushed to end of container, take up as much space as needed to display content/li>
  • align-items: flex-center; items centered in container on cross axis, only take up as much space as needed to display content

For aligning all items in a container.

to target single items, use align-self


.targetedItem(S) { 
    align-self: flex-start; 
}
Flex Direction Column

all the exact same properties as above, but note that the Main Axis is TOP TO BOTTOM and Cross Axis is LEFT TO RIGHT.

Wrapping

if you give an explicit width to the items in the container, it will scale down in size by default when the container is too small to display full size..

flex-wrap changes this behavior:


.container { 
    flex-wrap: nowrap; //DEFAULT, allows items to flex smaller in size as needed, but does not stretch larger.
    flex-wrap: wrap; //enforces explicit size... does not allow items to flex in size.  items will wrap to next row (cross axis) if container isnt wide enough.
}
Order

source order independance... using the order property, we can change the order of contained items, regardless of their order in the source code.


<div>
    <div class="container">
    <div class="item1">1 Home</div>
    <div class="item2">2 Search</div>
    <div class="item3">3 Logout</div>
</div>
                    
.item2 {
    order: 0; //DEFAULT VALUE
    order: 1; //would place this targeted item at the END of the flexbox regardless of order in html.  
              //At end, because 1 comes after any items with default value (0)
    order: -1; //would place this targeted item at the START of the flexbox regardless of order in html.  
               //At start, because -1 comes before any items with default value (0)
                    
}
